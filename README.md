# Payroll Management System in Java

A **console-based Payroll Management System** in Java that allows managing employee records, calculating salaries, and performing various payroll operations efficiently. It uses object-oriented programming principles, including inheritance and encapsulation, and provides a color-coded console interface for better readability.

---

## Features

- Add, edit, and delete employee records.
- Display all employees with detailed information.
- Calculate **gross salary** including allowances, overtime, leave allowance, and deductions like PF and professional tax.
- Search employees by ID.
- Sort employees by ID.
- Calculate **average salary** of all employees.
- Color-coded console output for better visibility.
- Enforces valid phone numbers and notice period rules.

---

## Technologies Used

- **Java SE** (Standard Edition)
- Object-Oriented Programming concepts:
  - Inheritance
  - Encapsulation
  - Polymorphism
- Console input/output using `Scanner`
